# Modbus TCP 數據中台與服務管理系統
> **系統定位**: 工業數據採集、通訊協定轉換與服務監控
> **核心技術**: Python, PySide6, nssm, Modbus TCP

---

## 📖 專案簡介
本專案提供一套工業級數據中台方案，旨在將多路異質通訊協定整合為標準 Modbus TCP 服務。系統支援掛載多個獨立服務實例 (例如 Port 500-512)，作為 PLC 採集端與 SCADA/Web 監控端之間的數據緩存層。

### 系統亮點
- **高可用性**: 透過 NSSM 守護進程，實現服務崩潰自動重啟。
- **隔離管理**: 每個服務實例擁有獨立日誌 (Log)，便於精確排錯。
- **即時監控**: 內置 PySide6 GUI 工具，支援數據變色提示與每秒自動刷新。
